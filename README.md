# AI Weather Intelligence Dashboard

An AI-powered weather dashboard built with Streamlit.

Features:
- Live weather data
- Temperature forecast charts
- Air quality data
- AI assistant for weather queries
- Voice-based weather questions

Tech Stack:
- Python
- Streamlit
- OpenWeather API
- Groq LLM
- Plotly

Run locally:

pip install -r requirements.txt
streamlit run ui/streamlit_ui.py
